Research Experiences

  • Object Detection for Autonomous Driving (Hitachi project)
    Sep 2019 - Apr 2020
    Researcher, the University of Michigan, Ann Arbor
    Autonomous Driving requires reliable models and hardware that maximizes inference performances. This project explores the possibilities of using heteogeneous systems (GPU, TPU, CPU, FPGA, etc) to fulfill the demands. My efforts are focused on TPUs. I modified, quantized, and compiled MobileNetv1 in tflite, deployed them in edgeTPUs and profiled the inference performance.
  • Software Defined Hardware (DARPA project)
    May 2019 - Apr 2020
    Researcher, the University of Michigan, Ann Arbor
    Designed test benches for TransPy, python library tailored for a reconfigurable processor. Reimplemented Naive Bayes SGD for sentiment analysis in C++, ran simulations, and tested the workflow with different configurations.
  • Runtime Neural Pruning
    May 2019 - Jan 2020
    Researcher, the University of Michigan, Ann Arbor
    Designed and implemented a prunable CNN framework with an RNN to dynamically prune neurons based on input complexity. Achieved ~ 5x speedup while maintaining an accuracy of ~90%.
  • Accelerating Variant Calling for Genome Sequencing
    Jan 2019 - Sep 2019
    Researcher, the University of Michigan, Ann Arbor
    Accelerated variant calling by making GATK multi-threaded and assigning PairHMM computations to FPGA. Developed a HOST-FPGA interface that dynamically adjusts the batch structure to minimize overheads and allows the HOST to control FPGAs in Java.
